Cuestiones técnicas

  • En el cliente la interacción se realiza enteramente por javascript, sin recargar la página, ni pedir html enteros al servidor
  • Mediante javascript se realiza reescritura e interpretación del URL. Con el método pushState de HTML5 y el evento popState, se puede navegar con los botones back y forward del browser
  • Los servicios provistos por el servidor, devuelven resultados en formato JSON. Todos estos servicios (obtener artista, CDs, busqueda, administracion, configuración, etc) se piden mediante AJAX

Iframe

  • Para realizar una búsqueda y obtener una página en modo iframe, el URL debe ser /index.php?iframe=true&busqueda=true¶metro=valor
  • La busqueda puede ser genérica (por cualquier campo), usando el parámetro buscar. Si no, puede usarse cualquiera de los parámetros que se muestran en la búsqueda avanzada
  • Ejemplos
  • index.php?iframe=true&showCD=1
  • index.php?iframe=true&busqueda=true&cancion=a
